Este proyecto combina detección de puntos faciales (landmarks) con reglas geométricas simples para identificar tres expresiones humanas comunes.
Está diseñado como demostración práctica de análisis facial en tiempo real, y puede servir como base para proyectos de:

Reconocimiento de emociones.
Control por gestos faciales.
Interfaces interactivas.
